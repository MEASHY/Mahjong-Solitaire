This directory of the 'Assets' folder is used for the storing of layouts.

The JSON file 'PackageList.json' contains the names of every layout package
available to the application. The names are to be stored in a jsonArray in
alphabetical order.

Each package directory in this folder contains a number of layouts as well as
a 'layouts.json'. This 'layouts.json' file will consist of a jsonArray
containing the names of every layout in the package, in alphabetical order.
