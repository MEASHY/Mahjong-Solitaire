This Directory of the 'Assets' folder is used for the storing of tilesets.

The JSON file 'SetList.json' contains the names of every tileset available
to the application. The names are to be stored in a jsonArray in alphabetical 
order.

Each tileset directory in this folder contains a number of tile images as well
as a tiles.json. This tiles.json file will contain a json object containing the 
details about the names of every tile, every selected tile, the visual effects,
as well as the number of tiles in the set.
