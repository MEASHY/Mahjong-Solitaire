findChildren() → {TileNode}

Finds the children of a TileNode and inserts them into the children array. As it finds the TileNode's children it also populates the parent array.

This function also propagates down the children finding all the children recursively When called on the layout roots this function will fill all children and the parents of the layout

setSpritePosition(numChildren)

Sets the sprite position of the TileNode to be centered on the screen in the correct layout position. The sprite is also scaled to the session scale parameter

Tile position is determined by the size of the tile and its position in the layout after determining its base position the position is further offset to be centered on its children

setTile(img)

Initializes a sprite with a given image for this tile at the origin. The img argument must specify a preloaded Phaser Sprite

We set the depth of the tile based on its position in the layout array. This ensures that each tile overlaps each other correctly This method works for 999x999x999 tiles
